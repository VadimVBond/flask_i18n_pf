# TASK_PROMPTS.md
Готовые команды для работы с AI

Эти команды помогают получать стабильный код.

---

## Продолжить код

Продолжи код с последней строки.
Не изменяй уже существующий код.

---

## Исправить ошибку

Найди и исправь ошибку.
Не переписывай файл полностью.

---

## Изменить функцию

Измени только указанную функцию.
Остальной код не трогай.

---

## Добавить функцию

Добавь новую функцию.
Не изменяй существующие функции.

---

## Добавить маршрут Flask

Добавь новый маршрут Flask.
Не изменяй существующие маршруты.

---

## Исправить шаблон

Исправь только шаблон.
Не изменяй Python-код.

---

## Улучшить код

Оптимизируй код.
Сохрани текущую архитектуру.

---

## Проверить код

Проверь файл на:

- дубли строк
- повторяющиеся импорты
- ошибки Flask
- ошибки маршрутов

Не изменяй код без необходимости.
