## Створення застосунку Flask

Кожен застосунок Flask має мати екземпляр класу `Flask`. Це WSGI-застосунок, якому сервер передає запити для подальшої обробки.

```python
from flask import Flask

app = Flask(__name__)
```

У першому рядку імпортуємо `Flask`, у другому — створюємо об’єкт застосунку та передаємо `__name__`, щоб Flask міг знаходити шаблони й статичні файли.

## Створення маршрутів

Маршрут пов’язує URL із функцією представлення.

```python
@app.route('/')
def index():
    return 'Hello World'
```

Коли застосунок отримує запит на `/`, Flask викликає `index()` і повертає результат.
